## Bi-Normal Sampling

```python
import matplotlib.pyplot as plt; plt.style.use('ggplot')
import numpy as np

mean = [0, 0]
cov = [[1, 0.5], [0.5, 0.5]]

x,y = np.random.multivariate_normal(mean, cov, 5000).T

plt.plot(x, y, 'o'); plt.axis('equal'); plt.show()
```
